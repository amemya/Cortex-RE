# Cortex-RE プロジェクト計画書

## プロジェクト概要
FC3S（マツダ RX-7 / 13Bロータリーエンジン）の機械式スロットルを電子制御（Drive-by-Wire / DBW）に置き換え、最終的には車両全体を制御するフルコン（Standalone ECU）へと発展させるプロジェクト。

## 構成予定パーツ
*   **MCU**: NUCLEO-H755ZI-Q (STM32H7シリーズ デュアルコア Cortex-M7/M4)
*   **モータードライバ**: TLE9201SGAUMA1 (SPI診断機能付きHブリッジ)
*   **スロットルペダル**: Mikuni APM30-105 (冗長センサー搭載)
*   **スロットルボディ**: Bosch 997 605 116 01 / 0 280 750 473 (冗長センサー搭載)

## フェーズ1: 電子スロットル (DBW) の実装

まずは純正ECU（または既存のエンジン制御系）と共存する、独立したDBWコントローラーとして開発する。

### 1. 安全性・フェイルセーフ設計（最重要事項）
電子スロットルの暴走は重大事故に直結するため、ASIL-D相当の安全設計を目指す。
*   **センサーの相互監視**: ペダル(APP)およびスロットル(TPS)の2系統センサー値の矛盾を常時監視。
*   **リンプホームモード**: 異常検知時は即座にモーター駆動を停止し、バネの力でスロットルを閉じる。
*   **物理カットオフ**: 異常時にマイコンのGPIOまたは独立監視回路からTLE9201のモーター駆動電源(12V)を物理的に遮断するリレー回路の実装。
*   **WDT**: マイコンフリーズ対策としてハードウェア・ウォッチドッグタイマーを実装。

### 2. ハードウェア・回路の要件
*   **電源ノイズ対策**: サージ対策（TVSダイオード、LCフィルター）、車載対応降圧スイッチングレギュレータ/LDOによるクリーンな電源供給。
*   **アナログ信号ノイズ対策**: ペダル・スロットルからのADC入力に対するRCローパスフィルターおよびオペアンプ等によるバッファ回路。
*   **純正ECUへのフィードバック**: 純正ECUにスロットル開度を伝えるためのDACまたはPWM+ローパスフィルター回路。
*   **将来を見据えた拡張**: フルコン化や他モジュールとの連携を見据え、初期段階から**CAN通信（CANトランシーバ回路）**を実装する。

### 3. メカニカル要件
*   FC3SのサージタンクにBoschスロットルを取り付けるための変換フランジアダプター作成。
*   ペダルマウント用ブラケット作成。

## フェーズ2: フルコン (Standalone ECU) への発展

NUCLEO-H755ZI-Qの圧倒的な処理能力（M7コアで3Dマップ計算やDBW制御、M4コアでクランク角割り込みと精密な点火・噴射タイミング制御）を活かし、エンジン制御全体を掌握する。

### 1. フルコン化に伴うメリット
*   DBWがアイドリングの空気量調整を自動で行うため、ISCV（アイドルスピードコントロールバルブ）が不要になる。

### 2. 追加が必要なハードウェア（インターフェース回路）
*   **クランク角センサー(CAS)入力**: FC3S純正のVRセンサー（アナログ正弦波）をデジタル方形波に変換する専用IC回路（MAX9924など）。
*   **各種センサー入力**: 吸気圧(MAP)、吸気温(IAT)、水温(CLT)用のアナログ入力回路。
*   **インジェクター駆動出力**: 大電流を流せるローサイド・スイッチング回路（パワーMOSFET等）。
*   **イグニッションコイル駆動出力**: 点火コイル用信号出力回路（スマートコイル化も検討）。
*   **ワイドバンドO2センサー入力**: 空燃比(AFR)フィードバック用のコントローラー（CJ125や外部CAN入力）。

### 3. ロータリーエンジン(13B)特有の制御ロジック
*   **点火制御**: リーディング(L)とトレーリング(T)の独立点火およびスプリット角の計算ロジック。
*   **燃料噴射制御**: プライマリーインジェクターとセカンダリーインジェクターのステージング（シームレスな切り替え・介入）制御。

## 開発ロードマップ
1.  **ベンチテスト (モックアップ)**: 机上でペダルとスロットルを接続し、PID制御でスロットルが追従する基本動作を確認する。
2.  **DBW専用基板の設計**: ノイズ対策やリレーを含めた車載用プロトタイプ基板を作成（CANトランシーバ含む）。
3.  **実車テスト (DBWのみ)**: 車載環境でのノイズ耐性、フェイルセーフ動作の確認。
4.  **フルコン回路の追加設計**: クランク角入力、インジェクター/点火コイル出力回路などの追加。
5.  **エンジン制御プログラムの実装**: rusEFIなどのオープンソースプロジェクトを参考にしつつ、13B特有の制御を実装。
