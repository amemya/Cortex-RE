# Cortex-RE

**FC3S (マツダ RX-7 / 13Bロータリーエンジン) の機械式スロットルを電子制御化 (Drive-by-Wire / DBW) し、最終的にフルコン (Standalone ECU) へと発展させるためのプロジェクト。**

## 概要

Cortex-REは、STM32H7シリーズの強力なデュアルコアマイコンを活用し、古い年式のスポーツカー（主にFC3S）のエンジン制御を現代的にアップデートすることを目標としています。
第一段階として独立した電子スロットルコントローラーの開発を行い、第二段階で点火・燃料噴射等を含むエンジン全体のマスター制御を行うフルコン（Standalone ECU）への拡張を目指します。

## 構成パーツ (ハードウェア)

*   **MCU**: NUCLEO-H755ZI-Q (STM32H7シリーズ デュアルコア Cortex-M7/M4)
*   **モータードライバ**: TLE9201SGAUMA1 (SPI診断機能付きHブリッジ)
*   **スロットルペダル (APP)**: Bosch 0 280 752 213 (冗長センサー搭載)
*   **スロットルボディ (TPS)**: Bosch 997 605 116 01 / 0 280 750 473 (冗長センサー搭載)
    *   [Bosch Motorsport Datasheet](https://www.bosch-motorsport.com/content/downloads/Raceparts/Resources/pdf/Data%20sheet_68749835_Electronic_Throttle_Body.pdf)

## 開発フェーズ

### フェーズ1: 電子スロットル (DBW) の実装
まずは純正ECU（または既存のエンジン制御系）と共存する、独立したDBWコントローラーとして開発します。
電子スロットルの制御は車両の安全性に直結するため、ASIL-D相当の安全基準を目標にフェイルセーフを最優先で設計します。

*   **安全性・フェイルセーフ設計 (最重要事項)**
    *   **センサー相互監視**: ペダル(APP)とスロットル(TPS)のそれぞれ2系統のセンサー値の矛盾を常時監視します。
    *   **リンプホームモード**: 異常検知時は即座にモーター駆動を停止し、バネの力でスロットルを安全に閉じます。
    *   **物理カットオフ**: 異常時にマイコンGPIOまたは独立監視回路から、モーター駆動電源(12V)を物理的に遮断するリレー回路を実装します。
    *   **WDT (ウォッチドッグタイマー)**: マイコンフリーズ対策をハードウェア機構として実装します。
*   **ハードウェア要件**
    *   電源・アナログ信号ノイズ対策 (TVSダイオード、LC/RCフィルター、オペアンプバッファ)
    *   純正ECUへのスロットル開度フィードバック回路（DAC / PWM+LPF等）
    *   拡張性を考慮したCAN通信（CANトランシーバ回路）の初期実装
*   **メカニカル要件**
    *   FC3Sサージタンク用にBoschスロットルを取り付けるための変換フランジアダプター作成
    *   ペダルマウントブラケット作成

### フェーズ2: フルコン (Standalone ECU) への発展
NUCLEO-H755ZI-Qの圧倒的な処理能力（M7コアで3Dマップ計算やDBW制御、M4コアでクランク角割り込みと精密な点火・噴射タイミング制御）を活かし、エンジン制御全体を掌握します。

*   **フルコン化のメリット**: DBWによるアイドルコントロール (ISCVの撤廃が可能に)。
*   **追加が必要なハードウェア（インターフェース回路）**
    *   **クランク角センサー(CAS)入力**: VRセンサー(アナログ正弦波)用デジタル変換回路 (MAX9924など)
    *   **各種センサー入力**: MAP(吸気圧), IAT(吸気温), CLT(水温)用のアナログ入力回路
    *   **インジェクター駆動出力**: 大電流用ローサイド・スイッチング回路（パワーMOSFET等）
    *   **イグニッションコイル駆動出力**: 点火コイル用信号出力回路 (スマートコイル化も検討)
    *   **ワイドバンドO2センサー入力**: 空燃比(AFR)フィードバック用のコントローラー (CJ125や外部CAN入力)
*   **ロータリーエンジン(13B)特有の制御ロジック**
    *   リーディング(L) / トレーリング(T)の独立点火およびスプリット角の計算ロジック
    *   プライマリー / セカンダリーインジェクターのシームレスなステージング（切り替え・介入）制御

## 開発ロードマップ

1.  **ベンチテスト (モックアップ)**: 机上でのペダル・スロットル接続とPID制御による基本追従の確認。
2.  **DBW専用基板の設計**: ノイズ対策やセキュリティリレー、CANトランシーバを含めた車載用プロトタイプ基板の作成。
3.  **実車テスト (DBWのみ)**: 車載環境でのノイズ耐性とフェイルセーフ動作の実証。
4.  **フルコン回路の追加設計**: クランク角入力、インジェクター/点火コイル出力回路などの追加デザイン。
5.  **エンジン制御プログラムの実装**: オープンソース（rusEFIなど）を参考にしつつ、13B特有の制御を実装。
